New in Linux Tools 0.10

Valgrind

Massif Charts generated by SWTChart

BIRT is no longer required to run the valgrind massif plugin. The chart displayed by massif is now generated using the SWTChart library, available at Orbit.

RDT Proxy

RDT Proxy

An library to leverage the development of remote execution of linuxtools plugins was added to linuxtools.
It provides a common layer to seamlessly access files and execute commands both locally and remotely. This was not integrated to any plugin yet, but it is a great step in converting plugins to work in remote environments.

Linux Tools Path

Linux Tools Path

It is now possible to have several releases of a linux tools installed in the system at the same time and select which will be used by Linux Tools plugins. In a project property it is possible to select a PATH that will be prepended to the System PATH before running commands. If the tool is not found in the selected path, the tool from system path will be used.
This feature is supported in:

  • Valgrind
  • gprof
  • gconv
  • oprofile
